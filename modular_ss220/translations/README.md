## Руководство по добавлению перевода

1. В директории `translations/public` создайте файл **`ru_names.toml`**.
2. Заполните его переводом, используя следующий формат:

```toml
["air alarm"]
nominative = "воздушная сигнализация"
genitive = "воздушной сигнализации"
dative = "воздушной сигнализации"
accusative = "воздушную сигнализацию"
instrumental = "воздушной сигнализацией"
prepositional = "воздушной сигнализации"
gender = "female"
```

Обязательными элементами являются название, записанное в `[""]`, которое должно полностью совпадать с названием из игры, а также поле `nominative`. Остальные поля заполнять необязательно. Если поле `gender` не указано, по умолчанию будет использоваться значение `male`.

После добавления перевода в созданный файл запустите задачу `split translation`:

1. Нажмите `Ctrl + Shift + P`.
2. Выберите `Tasks: Run Task`.
3. Выберите `split translation`.

В директории `modular_ss220/translations/public/ru_names` должны появиться новые файлы. Их необходимо добавить в ваш PR.

> [!TIP]
> Добавление перевода в `.toml`-фрагменты относится исключительно к `/obj/item/...`.
> Не нужно добавлять перевод объектов, которые не относятся к игровым предметам.
