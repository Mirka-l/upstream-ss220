## Руководство по добавлению перевода ##

В директории translation/public создайте файл ["ru_names.toml"], после чего заполните переводом который желаете добавить.

Пример формы заполнения:
	["air alarm"]
	nominative = "воздушная сигнализация"
	genitive = "воздушной сигнализации"
	dative = "воздушной сигнализации"
	accusative = "воздушную сигнализацию"
	instrumental = "воздушной сигнализацией"
	prepositional = "воздушной сигнализации"
	gender = "female"

Обязательными элементами является название, записанное в [""], оно должно быть абсолютно идентичным названию из игры, а так же (`nominative`). Остальное заполнять не обязательно, однако (`gender`) по стандарту будет указан как (`male`).

После добавления перевода в созданный файл, запустите скрипт нарезки (`split.cmd`) в директории (`./tools/translation`).
Файл (`ru_names.toml`) не должен быть добавлен в открытый вами ПР, после выполнения скрипта его нужно удалить.

## Запуск игры ##

Для проверки имеются 2 режима запуска, (`Build & DS Debug (+ TOML ru_names)`) для стандартного запуска через VSCode, и (`Build & DD Debug (+ TOML ru_names)`) для запуска через лаунчер byond. При первом запуске будет выполнена установка зависимостей.

> [!TIP]
> Добавление перевода в (`.toml`) фрагменты относится исключительно к (`/obj/item/..`)
> Не нужно добавлять перевод того, что не относится к игровым предметам
